# URL Shortener App

A simple URL shortener built with React.

## How to run

1. Install dependencies:
   ```
   npm install
   ```

2. Start the app:
   ```
   npm start
   ```

3. Open in browser at:
   ```
   http://localhost:3000
   ```

## Features
- Enter and shorten URLs
- View URL history with timestamps
- Copy shortened URLs to clipboard
- Clear URL history
